# 📉 Customer Churn Analysis (Python Project)

This beginner-friendly data analysis project explores **customer churn** behavior in the telecom sector using Python. The goal is to identify which customer groups are more likely to churn and what features influence their decisions.

---

## 🎯 Objective
To analyze customer data and uncover patterns in:
- Who is likely to churn (leave the service)
- What factors contribute to churn
- How churn relates to monthly charges, tenure, and demographics

---

## 🛠️ Tools & Libraries
- Python 3.12
- pandas (for data handling)
- seaborn & matplotlib (for visualization)

---

## 📁 Dataset
Dataset used: [Telco Customer Churn - Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

Filename: `WA_Fn-UseC_-Telco-Customer-Churn.csv`

---

## 📊 Key Insights
- 📈 **Churn Rate:** ~26.5%
- 🧓 Senior citizens have a higher churn rate.
- ⏳ Customers with **low tenure** and **high monthly charges** are more likely to churn.
- 📶 Contract type and payment method also influence churn risk.
